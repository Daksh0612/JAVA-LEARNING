What is an Array?

An array in Java is a collection of elements of the same type stored at contiguous memory locations. Arrays allow you to store multiple values in a single variable instead of declaring separate variables for each value.

1.Declaring an Array

int[] numbers = new int[5];  // Array of 5 integers

2.Initializing an Array

int[] marks = {90, 85, 70, 95, 88};

3.Accessing Array Elements

System.out.println(marks[0]); // prints 90
System.out.println(marks[3]); // prints 95

4.Looping Through an Array

int[] numbers = {1, 2, 3, 4, 5};
for(int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

5.Using Enhanced For Loop

for(int num : numbers) {
    System.out.println(num);
}
