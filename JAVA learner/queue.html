Q3. Implement Circular Queue using Array (Medium)

Create a circular queue class using a fixed-size array. Implement enqueue, dequeue, and isFull/isEmpty methods.

Waiting for your codeâ€¦
Solution â€” Circular Queue 
import java.util.*;
public class CircularQueue{
  int[] queue; int front=0, rear=-1, nItems=0, size;
  CircularQueue(int s){ size=s; queue=new int[s]; }

  void enqueue(int val){
    if(nItems==size) System.out.println("Full");
    else { rear=(rear+1)%size; queue[rear]=val; nItems++; }
  }

  int dequeue(){
    if(nItems==0) return -1;
    int temp=queue[front]; front=(front+1)%size; nItems--; return temp;
  }

  boolean isFull(){ return nItems==size; }
  boolean isEmpty(){ return nItems==0; }

  public static void main(String[] args){
    CircularQueue q=new CircularQueue(5);
    q.enqueue(10); q.enqueue(20); System.out.println(q.dequeue());
  }
}

Solution â€” Queue using 2 Stacks 
import java.util.*;
public class QueueUsingStacks{
  Stack s1=new Stack<>();
  Stack s2=new Stack<>();

  void enqueue(int val){ s1.push(val); }

  int dequeue(){
    if(s2.isEmpty()){
      while(!s1.isEmpty()) s2.push(s1.pop());
    }
    return s2.isEmpty()?-1:s2.pop();
  }

  public static void main(String[] args){
    QueueUsingStacks q=new QueueUsingStacks();
    q.enqueue(10); q.enqueue(20); System.out.println(q.dequeue());
  }
}
